# CastilloApplication
Aplicación con arquitectura hexagonal utilizando Spring Boot.


## Tecnologías utilizadas
-   Java 17+
-   Spring Boot
-   Maven

## Cómo ejecutar el proyecto

### 1. Clonar el repositorio

    git clone https://github.com/estefania227/CastilloApplication.git

### 2. Entrar al proyecto

    cd CastilloApplication
